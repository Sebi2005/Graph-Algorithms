# Graph-Algorithms


This repository contains implementations of fundamental graph algorithms, developed as part of my studies in Artificial Intelligence. The project focuses on understanding graph representations and solving classic problems using efficient algorithms.

🚀 Features
Graph implemented as a reusable class
Traversal algorithms:
Depth-First Search (DFS)
Breadth-First Search (BFS)
Shortest path algorithms:
Dijkstra’s Algorithm
Bellman-Ford Algorithm
Minimum Spanning Tree:
Prim’s Algorithm
Matching algorithms:
Kuhn’s Algorithm (Maximum Bipartite Matching)
Other:
Hamiltonian Cycle detection
📁 Project Structure
